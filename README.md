# 8-puzzle-game-buscas-largura-profundidade
Implementação do jogo 8-Puzzle em C, com suporte tanto para interação manual quanto para resolução automática utilizando algoritmos de busca em Inteligência Artificial.  O projeto permite ao usuário jogar ou solicitar que o sistema encontre a solução utilizando estratégias como Busca em Largura (BFS) e Busca em Profundidade Iterativa (IDDFS).
